What is this class?

In this dynamic and hands-on course, graduate students will delve into the cutting-edge world of modern infrastructure technologies. Emphasizing real-world applications, ISYS 53303 provides an in-depth exploration of advanced topics in infrastructure including: cloud computing (Infrastructure as a Service and Platform as a Service), identity management, Infrastructure as Code and evolving topics as appropriate each semester. The goal of the course is to equip students with the skills to design, implement, and manage robust and scalable IT infrastructure solutions

Throughout the course, students will:

  • Engage in practical exercises to design and implement various infrastructure solutions, focusing on scalability, security, and cost
  • effectiveness.
  • Critically analyze and evaluate different infrastructure technologies, understanding their place and purpose in various business contexts.
  • Apply theoretical knowledge to real
  • world scenarios, crafting detailed infrastructure strategies for hypothetical business cases.
  • Stay abreast of emerging trends and innovations in infrastructure technology, assessing their potential impact and integration into existing systems.
  • Explore the ethical and sustainable dimensions of technology deployment, with a focus on data privacy and environmentally responsible computing practices.
  • Gain hands
  • on experience with the latest tools and platforms in the field, enhancing their practical skill set.
  • Work collaboratively on team projects to solve complex infrastructure challenges, fostering skills in communication, teamwork, and innovation.

Course Objectives

Students in this course will gain a comprehensive understanding of cloud computing and its applications in modern IT environments. We will be using the Microsoft Azure platform to explore core concepts, services, and management strategies in cloud computing. By the end of the course, students will be able to:

  • Understand the core concepts of cloud computing and its significance in modern IT environments.
  • Gain a comprehensive understanding of cloud concepts and their applications in modern computing environments.
  • Explore cloud computing models, pricing structures, and service types to effectively utilize cloud resources.
Optionally, students may choose to take the Microsoft Azure Fundamentals certification exam (AZ-900) at the end of the course.

Course Outline

Core Concepts:
  • Define cloud computing and its significance in contemporary IT landscapes.
  • Explain the shared responsibility model and its implications for cloud users and providers.
  • Differentiate between public, private, and hybrid cloud models, identifying suitable use cases for each.
  • Analyze the consumption-based model and compare various cloud pricing strategies.
  • Investigate serverless computing and its role in optimizing resource utilization and scalability.
    Benefits of Cloud Services:
  • Examine the advantages of high availability, scalability, reliability, predictability, security, governance, and manageability in cloud environments.
    Cloud Service Types:
  • Define Infrastructure as a Service (IaaS), Platform as a Service (PaaS), and Software as a Service (SaaS), and outline their respective use cases.
    Azure Architecture and Services:
  • Explore core architectural components of Azure, including regions, availability zones, datacenters, resources, subscriptions, and management groups.
  • Investigate Azure compute and networking services, storage services, and identity, access, and security features.
    Azure Management and Governance:
  • Understand cost management strategies in Azure and utilize tools for optimizing cloud expenditure.
  • Implement governance and compliance measures using Azure Purview, Azure Policy, resource locks, and tagging.
  • Master the management and deployment of Azure resources through Azure Portal, Cloud Shell, Azure Arc, and Infrastructure as Code (IaC) techniques.
  • Monitor Azure resources effectively using Azure Advisor, Azure Service Health, and Azure Monitor tools.
